### PROJECT OVERVIEW

THIS IS A MODERN, MULTI-LANGUAGE MACRO TOOL DESIGNED WITH CUSTOMTKINTER. IT ALLOWS USERS TO AUTOMATE SIMPLE KEYPRESS AND MOUSE-CLICK SEQUENCES WITH A HIGHLY CUSTOMIZABLE INTERFACE.

### FEATURES

* **ONE-CLICK EXECUTION:** RUN THE TOOL DIRECTLY VIA THE .EXE WITHOUT INSTALLING PYTHON.
* **MULTI-LANGUAGE SUPPORT:** FULLY TRANSLATED INTO ENGLISH, ARABIC, FRENCH, AND ESPAÃ‘OL.
* **CUSTOMIZABLE DELAYS:** FINE-TUNE START DELAYS, ACTION DELAYS, AND FINAL DELAY TIMINGS.
* **HOTKEY INTEGRATION:** START THE MACRO INSTANTLY USING THE F6 KEY (OR YOUR OWN CUSTOM KEY).
* **MODERN UI:** SLEEK DARK MODE DESIGN FOR BETTER USER EXPERIENCE.

### HOW TO DOWNLOAD AND RUN

1. GO TO THE RELEASES SECTION ON THE RIGHT SIDE OF THIS PAGE.
2. DOWNLOAD THE PALIA_TICKET_TOUR.EXE FILE.
3. DOUBLE-CLICK THE FILE TO RUN IT.
4. **NOTE:** WINDOWS MAY FLAG THIS AS AN "UNKNOWN FILE." CLICK MORE INFO AND THEN RUN ANYWAY.

### HOW TO USE

* **SELECT LANGUAGE:** USE THE TOP-RIGHT DROPDOWN TO CHANGE THE INTERFACE LANGUAGE.
* **SET DELAYS:** ADJUST THE SECONDS TO MATCH YOUR SYSTEM SPEED.
* **SET KEYS:** CHOOSE WHICH KEYS (K1/K2) THE MACRO SHOULD PRESS.
* **START:** CLICK THE START MACRO BUTTON OR PRESS YOUR HOTKEY.
* **STOP:** CLICK THE STOP BUTTON AT ANY TIME TO END THE PROCESS.
