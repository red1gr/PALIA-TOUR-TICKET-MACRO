### PROJECT OVERVIEW

THIS IS A MODERN, MULTI-LANGUAGE MACRO TOOL BUILT WITH CUSTOMTKINTER. IT ALLOWS USERS TO AUTOMATE SIMPLE KEYBOARD AND MOUSE ACTIONS THROUGH A CLEAN AND HIGHLY CUSTOMIZABLE INTERFACE.

THIS TOOL HELPS AUTOMATE VISITING HOUSES IN PALIA AND INTERACTING WITH THEM MORE QUICKLY TO OBTAIN TOUR TICKETS.

---

### FEATURES

* **ONE-CLICK EXECUTION:** RUN THE TOOL DIRECTLY AS AN `.EXE` FILE WITHOUT INSTALLING PYTHON.
* **MULTI-LANGUAGE SUPPORT:** AVAILABLE IN ENGLISH, ARABIC, FRENCH, AND SPANISH.
* **CUSTOMIZABLE DELAYS:** ADJUST START DELAYS, ACTION SPEEDS, AND FINAL WAIT TIMES.
* **HOTKEY SUPPORT:** START THE MACRO INSTANTLY USING THE F6 KEY (OR A CUSTOM KEY).
* **MODERN UI:** CLEAN DARK MODE INTERFACE FOR A BETTER USER EXPERIENCE.
